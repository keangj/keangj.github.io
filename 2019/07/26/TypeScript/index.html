TypeScript

TypeScript 是 JavaScript 的超集

TypeScript 支持的类型

数组

枚举

函数重载

any

any 可以是任何值

元组

固定长度数组

never

不存在的值

Unknown

和 any 一样，所有的值都可以赋给 unknown。

但 unknown 只能被赋值为 unknow 和 any

unknown 不可以直接被操作

void

没有类型

断言

类型断言

断言值为某种类型

非空断言

value! 断言当前值不为 null 和 undefined

接口

接口用来描述一个对象拥有哪些属性

只读属性

可选属性

接口继承

传 interface 未定义的属性

  • 类型断言

  • 索引签名

高级类型

交叉类型

多个类型合并为一个类型

多类型同属性合并，类型为 never

联合类型

值可以是定义的某一种类型

类型别名

为类型起一个新的名字

字面量类型

this 类型

指定 this 的类型

索引类型

Readonly

返回 T 并将所有属性的类型变为只读

ReadonlyArray

只读的数组

Required

返回 T 并将类型所有变为 required

Partial

返回 T 并将类型所有变为可选

Omit<T, K>

返回 T 并去除 K 属性

Pick<T, K>

返回 T 中对应的 K 属性

Exclude<T, U>

NonNullavle

keyof

extends

typeof

可识别类型

1.有一个共有的字段
2.共有字段是可穷举的
