TypeScript

TypeScript 是 JavaScript 的超集

TypeScript 支持的类型

数组

枚举

也可以为枚举赋值

如果从中间开始赋值为字符串需要重新为之后的成员赋初始值

计算成员

常量枚举

函数

函数重载

any

any 为顶级类型（top type），可以是任何值

Unknown

和 any 一样是顶级类型，所有的值都可以赋给 unknown。

但 unknown 只能被赋值为 unknow 和 any

unknown 不可以直接被操作

never

never 和 any unknown 相反为底部类型（bottom type），不存在的值

void

没有类型

元组

固定长度数组

断言

类型断言

断言值为某种类型

非空断言

value! 断言当前值不为 null 和 undefined

接口

接口用来描述一个对象拥有哪些属性

只读属性

可选属性

接口继承

循环引用自身

implements

从 class 或 interface 实现所有的属性和方法，同时可以重写属性和方法

implements 和 extends 的区别

传 interface 未定义的属性

  • 类型断言

  • 索引签名

interface 和 type 区别

  1. interface 使用 extends 继承，type 使用 & 联合类型继承
  2. interface 可以通过重复声明来拓展，type 同名只能声明一次
  3. interface 会创建新的类型，type 只是类型别名
  4. Interface 可以循环引用自身，type 不可以

高级类型

交叉类型

多个类型合并为一个类型

多类型同属性合并，类型为 never

联合类型

值可以是定义的某一种类型

类型别名

为类型起一个新的名字

字面量类型

this 类型

指定 this 的类型

索引类型

可识别类型

1.有一个共有的字段
2.共有字段是可穷举的

条件类型

泛型

工具类型

ReturnType

返回 T 的返回值类型，T 为 (…args: any) => any

Record<K, T>

定义对象的 key value 类型

Readonly

返回 T 并将所有属性的类型变为只读

ReadonlyArray

只读的数组

Required

返回 T 并将类型所有变为 required

Partial

返回 T 并将类型所有变为可选

Omit<T, K>

返回 T 并去除 K 属性

Pick<T, K>

返回 T 中对应的 K 属性

Exclude<T, U>

返回 T 中不包含 U 的

Extract<T, U>

返回 T 和 U 的交集

NonNullable

返回 T 并去除 undefined

infer

keyof

获取类型的所有 key

extends

泛型约束

in

包含某属性

typeof

可以用来获取变量的类型

as const

其他

在 window 上添加变量声明
