# 交易

## 指标

### 均线(MA)

简单移动平均线(Moving Average)

均线是根据过去的价格走势绘制出的

价格走势决定均线走势，均线滞后于价格

周期为 N 的移动平均线 = N 根 K 线收盘个之和 / N

1. 均线**描述过去**的走势不是预测未来
2. 均线跟随价格，描述的是价格上涨的节奏
3. 判断趋势有更高效的方式

### SMA 简单移动平均线

N 周期的价格直接平均相加

### EMA 指数移动平均线

近期的价格分配更高的权重（对价格敏感度更高）

价格在 EMA 上方，为多头信号。

价格在 EMA 下方，为空头信号。

### MACD 指数平滑异同移动平均线

DIF 快线 = 12EMA - 26EMA

快线的数值等于两根均线的**差离值**

DEA 慢线 = 快线度移动平均线

能量柱 = n* （快线 DIF - 慢线 DEA）

**指标应用**

- 根据快线 DIF 位置判断趋势。

  快线上穿 0 轴，在 0 轴上方为多头趋势。

  快线下穿 0 轴，在 0 轴下方为空头趋势。

- 金叉死叉作为多空信号

  快线上穿慢线形成**金叉**，作为多头信号。

  快线下穿慢线形成**死叉**，作为空头信号。

- 快线位置和交叉信号

  当快线在 0 轴上方为多头趋势，0 轴上方又出现了快线上穿慢线的金叉作为多头信号。

  当快线在 0 轴下方为空头趋势，0 轴下方又出现了快线下穿慢线的死叉作为空头信号。

- 指标背离判断行情反转

  背离指的是价格连续的上涨，不断的创新高。但是 MACD 指标快线的峰值在减少。就形成了指标和价格的背离，背离往往被作为判断行情可能会反转的信号。

- 差离值的正负

  小周期的均线受当前价格走势的影响较大更加贴近价格，大周期的均线会相对滞后。当价格连续上涨，小周期的均线会在大周期均线的上方，两根均线的差离值为正，快线的数值大于 0 在 0 轴的上方。

- 差离值的变化

  影响两根均线差离值大小的因素是价格上涨的速度。价格上涨速度快，价格在单位时间内的变化更大，短周期均线受当前价格的影响更大反应的速度更快，长周期均线会相对会更加滞后，就会使得两根均线的差离值变大；当上涨的速度减缓或转为下跌，两根均线的差离值就会减小。价格涨跌速度的变化就影响了两根均线差离值的变化，也就影响了 MACD 快线数值的变化。

  MACD 的金叉死叉信号是受到价格涨跌速度的变化或者涨跌的反转所影响的。

- 指标的背离

  背离描述了上涨的速度在减缓。

  上涨的幅度和上涨的时间，上涨的时间相近但是上涨的幅度在减小，或者上涨的幅度相近但是上涨近过了更长的时间。这种情况都会形成指标和价格的背离。

### BOLL

指标公式（默认参数 20）

中轨线 = N 日移动平均线

上轨线 = 中轨线 + 2 倍的标准差

下轨线 = 中轨线 - 2 倍的标准差

2 倍的标准差可以覆盖 95% 的行情

数据越稳定价值越大

指标计算

1. 计算 MA

   近期 20 根 K 线收盘价相加求和再 ÷ 20

2. 计算标准差 MD

   方差= 20 根 K 线（收盘价 - MA )平方之和 ÷ 20

   标准差 MD = 平方根方差

3. 计算上下轨

   中轨 = MA

   上轨 = MA + 2MD

   下轨 = MA - 2MD

指标用法

- 支撑阻力

  上轨对价格有压力作用

  下轨对价格有支撑作用

- 超买超卖

  上涨超过布林带上轨的行情叫超买。

  下跌超过布林带下轨的行情叫超卖。

- 判断趋势

  布林指标是基于 20 周期均线计算而来的，均线会随着价格的涨跌发生变化，根据布林带的斜率可以判断当下的趋势是空头还是多头。

- 价格上涨 -> 价格和均线离散程度增大 -> 标准差增大 -> BOLL 开口
