# Linux 命令行



``` sh
mkdir <dirName> # 创建文件夹
touch <fileName> # 创建文件
vi <fileName> # 使用 vim 编辑文件
cat <fileName> # 查看文件内容
cp -r /test/[<fileName> | *] /test2 # 将 test 的文件复制到 test2
mv -r /test/[<fileName> | *] /test2 # 将 test 的文件移动到 test2
rm -rf <fileName> # 删除文件
pwd # 查看当前目录路径
kill -9 <pid> # 杀死进程
```
